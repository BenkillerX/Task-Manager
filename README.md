# Task Manager

A simple and clean **Task Manager Web Application** built using **HTML, CSS, and JavaScript**.
This app allows users to manage their daily tasks by adding, editing, completing, and deleting tasks.

##  Features

* Add new tasks
* Edit existing tasks
* Mark tasks as completed
* Delete tasks
* Clean and simple user interface
* Dynamic task rendering with JavaScript

##  Technologies Used

* HTML
* CSS
* JavaScript (Vanilla JS)
* DOM Manipulation
